# turso::transaction

## Module transaction

Module for transaction support in the Turso database client.

### Structs

- **Transaction**  
  Represents a transaction on a database connection.

### Enums

- **DropBehavior**  
  Options for how a Transaction should behave when it is dropped.

- **TransactionBehavior**  
  Options for transaction behavior. See [BEGIN TRANSACTION](http://www.sqlite.org/lang_transaction.html) for details.
